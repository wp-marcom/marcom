WestPressInventory Settings Table

Setting Description
Purchaser Order Accepted Notification This feature will send an Email Notification to the Purchaser when the order that was placed has been accepted by the Sales or Supplier.
Requisition Approved Notification Sends the Requisitioner an email notification that their Requisition has been approved. This is triggered by the Purchaser using the "Approve Selected Requisitions" choice on the Process Requisitions page.
Requisition Denied Notification Sends an email notification to the Requisitioner when a Req is denied. Also includes the description of denial, if entered.
Downsampling The MarcomCentral Low Resolution Proofing feature downsamples uploaded template and images assets to deliver a 72dpi proof in the Portal.
Publish Template Override Allows the Manager User to Publish a template even though outstanding orders using the previously published version of the template exist. Note: This should be used with caution -- if the new version of the template being Published does not have the same variables as the previous one used in outstanding orders, you will be unable to process those orders.
Retain Form Data on Timeout Automatically saves the information filled in to a product form by the portal user in the event that they timeout prior to adding the product to the cart.
Remove Flash Upload in Admin Tool This feature removes Flash from being used in the Admin Tool for Library Image uploads and Template uploads. It should be enabled if admin users are using a browser version which no longer supports Flash.
New Portal Catalog: Skip Versioned Product Details page For New Portal only. When active, Users will go directly to the customize product page for Versioned products when clicking the link in the catalog or in search results.
New Portal Catalog: Increase Folder Display For New Portal only. Expands the maximum number of folders visible on catalog page to 16. The default display is only 4.
New Portal login redirect When active, users clicking on classic portal URLs will be redirected to the New Portal login page when New Portal is enabled.
Hide Self Registration Link on Login Hides the Self Registration link on the Login page.
